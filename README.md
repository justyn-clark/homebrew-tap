# Homebrew Tap for SMALL CLI

This is the official Homebrew tap for the SMALL CLI.

## Installation

```sh
brew tap justyn-clark/tap
brew install small
```

## Upgrade

```sh
brew upgrade small
```

## Uninstall

```sh
brew uninstall small
brew untap justyn-clark/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| small   | SMALL CLI - Command line interface for the SMALL protocol |
