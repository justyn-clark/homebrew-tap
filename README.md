# Homebrew tap for Justyn Clark CLI tools

This is the official Homebrew tap for published JCN command-line tools.

## Installation

```sh
brew tap justyn-clark/homebrew-tap
```

Install a formula:

```sh
brew install small
# or
brew install devports
```

## Upgrade

```sh
brew upgrade small
brew upgrade devports
```

## Uninstall

```sh
brew uninstall small
brew uninstall devports
brew untap justyn-clark/homebrew-tap
```

## Available Formulae

| Formula  | Description |
|----------|-------------|
| small    | SMALL CLI - Command line interface for the SMALL protocol |
| devports | CLI for managing local development ports and configured service URLs |
